# Hackandslash

Simple isometric hack and slash game. Currently early experimental version.

![screenshot](http://laastine.kapsi.fi/kuvat/hackandslash.gif)

## Build
```
cargo install
cargo run
```

## Controls

`WASD`

Tested with Rust 1.13
